I manipulate the Document Object Model (DOM) to interact with the structure of web documents. It allows me to dynamically change the content, structure, and style of a webpage through JavaScript.
I select an element from the DOM using methods like `document.getElementById()`, `document.querySelector()`, or `document.getElementsByClassName()`. These methods help me access and manipulate specific elements in the document.
I use event delegation to add a single event listener to a parent element instead of individual listeners for each child element. This technique improves performance and simplifies event management for dynamically added elements.
I manipulate an element's attributes using methods like `setAttribute()` and `getAttribute()`. To change styles, I access the `style` property of the element and set CSS properties directly, for example, `element.style.color = 'red';`.
